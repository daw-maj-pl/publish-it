## PublishIt

Application for publishing interesting posts

### Project Specifications

- CRUD operations
- Ability to comment posts
- Authentication on the server
- Animations

### Technologies

- Next.js 13
- NextAuth.js
- TypeScript
- PostgreSQL
- Prisma
- TanStack Query/React Query
- TailwindCSS
- Framer Motion
- React Hot Toast

[Go to live demo](https://publish-it-pi.vercel.app)
